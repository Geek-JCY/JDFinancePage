## 项目介绍

这是一个高仿京东金融的项目，在熟悉了原项目的构造布局之后，进行的页面重构。

### 技术亮点
`组件化设计` 和 `css模块化`+`组件化`的设计思想

### 技术应用

```markdown
### 开发环境
	Node

### 开发工具
	Visual Studio

### 构建工具
	webpack

### 基础技术框架
	vue + vue-router + sass

### 页面自适应
	rem + hotcss

### 代码格式化
	eslint + vue-style-loader

### 版本控制器
	git
```
